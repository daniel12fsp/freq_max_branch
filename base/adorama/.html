About the Acer C720-2420

The Acer C720-2420 11.6" HD Chromebook Computer (Granite Gray) features the innovative Google Chrome OS operating system. Google Chrome OS is a Linux-based operating system designed by Google to work exclusively with web applications. This Acer Chromebook combines the aspects of a pure cloud client with the hardware of a traditional laptop, which puts it in its very own class of portable notebook computers.

Paying homage to Google Chrome's minimalist appearance, this Acer Chromebook has a simple and clean look with a granite gray finish. The Chrome OS keyboard features Chrome OS Function keys, which can directly control the browser, volume and screen brightness. The touchpad features click-anywhere functionality and supports one-finger touch to left-click, two-finger touch to right-click and two-finger scrolling.

The 11.6" widescreen display is backlit with LED lamps for better image quality and energy efficiency. The screen has a 16:9 aspect ratio as well as a 1366 x 768 native resolution, thanks to the integrated Intel HD Graphics. The screen also has an antiglare finish so the Chromebook can be used in brightly-lit environments.

The system is powered by a dual-core 1.4 GHz Intel Celeron 2955U processor. It also has 2GB of DDR3L RAM, a 32GB solid state drive, a Secure Digital media card reader, 10/100 Mb/s Fast Ethernet, 802.11a/b/g/n Wi-Fi, an integrated webcam and microphone as well as built-in stereo speakers. Chrome OS is the installed operating system.

Chrome OS
Using Linux as a base, Google designed Chrome OS to work exclusively with web applications. The user interface takes a minimalist approach, which greatly resembles the Google Chrome web browser. Chrome OS was built for people who spend a majority of their time on the computer connected to the Internet. The only application that comes installed on Chrome OS is a special Google Chrome web browser with an integrated media player and file manager. All other applications are accessed by adding apps from the Google Chrome Web Store.

Wireless Connectivity
Geared for portable use, you'll be able to stay online at home, outdoors or on your morning commute with built-in 802.11a/b/g/n Wi-Fi. 802.11a/b/g/n Wi-Fi support ensures that you'll always be able to jump on the Internet to access your email, browse the Web or connect with your social networks.

Dual-Core 4th-Gen Haswell Intel Celeron Performance
The Acer Chromebook houses a 4th-generation Haswell Intel Celeron 2955U processor. Its dual cores can reach a stock speed of 1.4 GHz. The Haswell processor also uses low voltage for improved battery life without sacrificing performance. With a dedicated 2MB L3 cache and 2GB of DDR3L RAM, you'll be able to perform tasks efficiently.

Chrome Web Apps
This Acer Chromebook has the ability to run millions of different web apps, from games to spreadsheets to photo editors. Thanks to the power of HTML5, many web apps can keep working even in the rare moments of when you're not connected to the Internet. Visit the Chrome Web Store to try out the latest web apps or just type in a URL address. No installation CDs required.

Google Drive
Store and access your photos, music, videos, documents and more from anywhere with Google Drive. All your files are backed up automatically online safely and securely.

12 Free Gogo Inflight Passes
The Acer Chromebook comes with 12 free Gogo Inflight passes so you can stay online even when you're not on the ground. They have an estimated $150 value and are redeemable for up to two years after date of purchase. Please note that Gogo Inflight passes are only usable on continental U.S. flights of participating airlines.

Seamless Updates
Over time, your Acer Chromebook will get faster and more efficient. Unlike a traditional computer, your Acer Chromebook updates itself automatically upon booting up. All of your web apps stay up-to-date and you will always have the latest and best version of the Chrome OS operating system without having to do it manually. Annoying update prompts are not included since all updates are done in the background seamlessly.

Multi-Layered Security
This Acer Chromebook has the first consumer operating system that was designed from the ground up to defend against the ongoing threat of malware and viruses. Chrome OS employs the principle of "defense in depth" to provide multiple layers of protection, including sandboxing, data encryption and verified boot.
