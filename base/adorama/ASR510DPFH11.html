About the Asus R510DP-FH11

The R510DP-FH11 15.6" Notebook Computer from ASUS features a dark gray design with a concentric circle finish. The 15.6" widescreen display features LED backlight technology. The screen has a 16:9 aspect ratio and a Full HD 1920 x 1080 native resolution. The resolution allows the ASUS notebook to play back 720p HD content without upscaling. The display also has a matte finish to help reduce glare.

The system is powered by a quad-core 2.5 GHz AMD A10-5750M processor. It also has 8GB of DDR3 RAM, a 750GB 5400 rpm hard drive, a SuperMulti DVD burner, a multi-format media card reader, 10/100/1000 Mbps Gigabit Ethernet, 802.11b/g/n Wi-Fi, Bluetooth 4.0, an integrated webcam and microphone. There are also built-in stereo speakers with SonicMaster audio technology.

Windows 8 (64-bit) is the installed operating system. Windows 8 replaces the traditional Start menu with a redesigned Start screen. The Start screen features tiles, which provide notifications in real-time. Windows 8 also focuses more on cloud storage as well as apps, which can be downloaded from the Windows Store. Windows 8 provides a unique user experience that differs from previous versions of Windows.

Quad-Core AMD A10 Performance
The ASUS notebook houses an AMD A10-5750M processor. Its quad cores can reach a stock speed of 2.5GHz. The processor can also be overclocked up to 3.5GHz, thanks to AMD TurboCore technology 3.0. With a dedicated 4MB L2 cache and 8GB of DDR3 RAM , you'll be able to multitask quickly and efficiently.

15.6" Antiglare LED-Backlit Display
The 15.6" widescreen display features LED backlight technology and utilizes a matte screen finish for reduced glare. With a 16:9 aspect ratio and a Full HD 1920 x 1080 native resolution, the display can playback 1080p Full HD content without upscaling.

AMD Radeon HD 8670M Graphics
This ASUS notebook comes with a dedicated AMD Radeon HD 8670M graphics card with 2GB of discrete memory. This integrated graphics chip supports Unified Video Decoder 4.01 for hardware based video decoding, OpenCL 1.2 and OpenGL 4.2 graphics standards, and DirectX 11.1 for high performance GPU-accelerated tessellation.

Connectivity
For Internet access, you have a choice of using the built-in 10/100/1000 Mbps Gigabit Ethernet RJ-45 jack or go wireless with 802.11b/g/n Wi-Fi. You can use Bluetooth 4.0 to wirelessly connect to all your peripherals, such as a wireless mouse or keyboard. The HDMI port allows you to output video to an HDTV.
